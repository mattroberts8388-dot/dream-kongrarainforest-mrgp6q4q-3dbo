# Kongra Rainforest Ecosystem

A Fabric mod for Minecraft **1.20.1** that turns the jungle into a deadly rainforest ecosystem.

## What it does

- **Rain hurts you.** If you stand out in the rain (under open sky) for too long, you will begin to lose health. The longer you stay exposed, the harder the rain hits.
- **Rainforest animals** spawn in jungle biomes: the **Rainforest Jaguar** (aggressive predator), the **Rainforest Toucan** (harmless flyer), and the **Rainforest Tapir** (skittish forager).
- **KONGRA** – the Big Boss. Body of a gorilla, head of a king cobra. 200 HP, poisons and weakens anyone he bites. He drops the rare **Kongra Scales** and **Venom Glands** needed for the armor.
- **KONGRA Armor** (camouflage neon-green / black / brown) is the only thing that fully protects you from the deadly rainforest rain — and gives you a fighting chance against KONGRA himself. It is crafted from the hardest-to-find rainforest materials: Kongra Scales, Venom Glands, and Rainforest Resin (dropped by tapirs).

Wear the **full set** (helmet + chestplate + leggings + boots) to become immune to the rain damage.

## How to build the mod (NO Java install needed — GitHub builds it for FREE)

You do **not** need to install Java or any development tools. GitHub will compile the `.jar` for you in the cloud.

1. **Create a GitHub account** at https://github.com (free).
2. Click the **+** in the top-right → **New repository**. Give it any name, keep it Public, and click **Create repository**.
3. On the new repository page, click the **"uploading an existing file"** link.
4. **Extract the ZIP** you downloaded (double-click it) so you have a normal folder.
5. **⚠️ macOS users — IMPORTANT:** The `.github` folder is **invisible** in Finder by default. If you don't include it, the build will never run! In Finder, press **Cmd + Shift + .** (period) to reveal hidden files. You will now see the `.github` folder.
6. Open the extracted folder and select **ALL** files and folders **INSIDE** it (including the hidden `.github` folder). Do **not** drag the outer folder — drag its **contents**.
7. Drag all of those selected files/folders into the GitHub upload area.
8. Scroll down and click **Commit changes**.
9. Click the **Actions** tab at the top of your repository. You'll see a build running.
10. Wait about **2 minutes** for it to finish (green check mark).
11. Click the completed build, then scroll down to **Artifacts** and download **mod-jar**.
12. Unzip it to get the `.jar`. Copy that `.jar` into your `.minecraft/mods/` folder.

You also need **Fabric Loader** and the **Fabric API** installed for Minecraft 1.20.1.

Enjoy the rainforest — and beware KONGRA.
